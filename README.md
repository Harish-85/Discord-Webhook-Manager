# A tool to manage multiple discord webhooks between multiple servers and channels

A GUI written using qt and C++ so that I can manage all my webhooks in one place

## How to use
Click the edit button on top and then setup your webhooks in the config window and press 'ok' to save
<img width="875" height="419" alt="image" src="https://github.com/user-attachments/assets/fe1a4be2-3bfa-4b67-a155-52c648527a3e" />

Now set your target server, channel and webhook
<img width="557" height="411" alt="image" src="https://github.com/user-attachments/assets/5f885c79-e9f1-4e75-a83e-afb0558244f0" />

Then press send to send the message
